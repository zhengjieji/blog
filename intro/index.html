Web安全学习笔记 - HTTP协议

Posted on 2022-07-03 | In Web Security

综述

超文本传输协议(HTTP)是一个用于传输超媒体文档的应用层协议。它是为Web浏览器与Web服务器之间的通信而设计的，但也可以用于其他目的。

工作原理

  • 客户端连接到Web服务器
  • 发送HTTP请求
  • 服务器接受请求并返回HTTP响应
  • 释放连接TCP连接
  • 客户端浏览器解析HTML内容

请求方法 - GET 和 POST

Web安全学习笔记 - Web前端 - HTML、CSS、JavaScript

Posted on 2022-07-03 | In Web Security

综述

HTML，CSS和JavaScript是三种前端所使用的主要语言。

HTML

超文本标记语言(HTML)是一种用于创建网页的标准标记语言。

HTML的网页结构

HTML版本声明：告诉浏览器HTML的版本
HTML头部部分：涉及页面标题、字符集、样式、链接
HTML主体部分：浏览器展现给用户的内容

HTML的标签/元素/属性

标签：HTML页面又标签和内容组成
元素：开始标签到结束标签之间的代码
属性：一般描述于开始标签中

CSS

层叠样式表(CSS)，是一种 样式表 语言，用来描述 HTML 或 XML文档的呈现。

JavaScript

JavaScript是可以插入HTML页面的轻量级编程语言。Node.js简单而言是运行在服务端的JavaScript。
