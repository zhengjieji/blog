Web安全学习笔记

Posted on 2022-07-03 | In Web Security

综述

Web安全技术旨在保护系统、网络、程序和数据免受网络风险，例如网络攻击、破坏或未经授权的访问。Web服务组件从表层到底层一共有9层，分别为：Web浏览器，第三方内容，Web前端框架，Web应用，Web开发框架，Web服务端语言，Web容器，存储，操作系统。我们需要学习数据是如何在各层组件中被处理的，从而避免可能出现的安全问题。学习Web安全技术需要基本的计算机知识和编程技能。需要掌握的编程语言有：PHP，Java，Python, SQL。
